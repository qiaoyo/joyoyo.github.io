several coding methods of SNN

1. rate coding(Poisson coding)

​ 最常用的编码方式。将每一个输入像素转换为脉冲序列，比较典型的一种是泊松分布的脉冲序列。参数只有一个啦，就是λ，根据图像的灰度值算出。然后一些预设的参数包括time，dt。在bindsnet，brian2的源码中都找到了possionspike的源码。

def poisson(
    datum: torch.Tensor, time: int, dt: float = 1.0, device="cpu", **kwargs
) -> torch.Tensor:

    rate = torch.zeros(size, device=device)
    rate[datum != 0] = 1 / datum[datum != 0] * (time / dt)
    #根据概率产生脉冲序列
    dist = torch.distributions.Poisson(rate=rate)
    intervals = dist.sample(sample_shape=torch.Size([time + 1]))
    intervals[:, datum != 0] += (intervals[ :, datum != 0] == 0).float()
    times = torch.cumsum(intervals, dim=0).long()
    spikes = torch.zeros(time + 1, size, device=device).byte()
    spikes[times, torch.arange(size)] = 1

​ population rank order coding

​ 对于每个神经元，构造一个GRF（高斯感知域）来编码它。这个GRF中包括p个神经元，每个神经元又有一个 $$ (\mu_{ij},\sigma_{ij}) $$

$$ A_{ij}=e^{-\frac{(s_i-\mu_{ij})^2}{2*\sigma_ij^2}} $$
